<언어> 
1) HTML (난이도 쉬움) (Hyper Text Markup Language) (h1~h6까지) (숫자가 작을수록 글자가 큰글자)
페이지의 제목, 문단, 표, 이미지, 동영상 등 웹의 구조를 만들어주는 언어 

HTML 5 - 상단,하단 페이지에 의미를 부여하는 규칙?

2) CSS (난이도 최상) (Cascading Style Sheets) - 레이아웃(페이지 자리잡는 구조) - (구조 잡는 것이 어렵지만 이해만 잘하면 됨) - 
                       (시각적으로 보여지는 것 크기가 큰지 작은지), 

실제 화면에 표시되는 방법(색상, 크기, 폰트, 레이아웃 등)을 지정해 콘텐츠를 꾸며주는 시
각적인 표현(정적)

3) JS (JavaScript) ( 페이지가 화려해짐) (개발자들이 콘텐츠들을 바꿀수도 있음) 
콘텐츠를 바꾸고 움직이는 등 페이지를 동작시키는 동적 처리 (움직임을 잡기위해 필수사용)

4) 웹 표준 (Web Standard)이란 ‘웹에서 사용되는 표준 기술이나 규칙’을 의미,

5) 크로스 브라우징 (여러 화면에서 동일하게 표현되고 보여지는 방법) (크롬, 엣지, 사파리, 오페라 , 파이어폭스 사이트에서 다르게 나오지 않고 똑같이 나오는게 하는것) 

크로스 브라우징(Cross Browsing)이란 조금은 다르게 구동되는 여러 브라우저에서, 동일한 사용자 경험(같은 화면, 같은 동작 등)을 줄 수 있도록 제작하는 기술, 방법.

6) 웹 접근성

웹 접근성이란 고령자, 장애인 같은 신체적, 환경적 조건에 제한이 있는
사용자를 포함한, 모든 사용자들이 동등하게 접근할 수 있는 웹 콘텐츠를 제작하는 방법

7) 화면 구조
1.창(전체)  2.탭(상단)  3.주소창(주소 입력창) 4.뷰포트(보여지는 화면)
